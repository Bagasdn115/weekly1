Version Control and Branch Management (Git)

Versioning adalah mengetur versi dari Source Code program. Tools yang digunakan dalam menggukan git adalah Version Control System (VCS), Source Code Manager(SCM) dah Revision Control System(RCS). Git adalah salah satu version control system popule yang digunakan para developer untuk mengembangkan software sacara bersama-sama
